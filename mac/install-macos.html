Instando em um disco inteiro


Para que todo o disco sejá usado devemos usar o "Disk Utility - Utilidade de Disco" para formatar o disco e instalar além de formatar para uso da instalação do MacOS


O disco deve ser formatado em GPT

Formatamos os disco em APFS pois se trata do novo formato da Apple para seus produtos novos*


Devemos agora instalar o MacOS no Disco que foi Formatado para uso do sistema































































Para instalaçao em volume para DUAL-BOOT (Windows - MacOS)


Paa instalar o MacOS + Windows devemos primeiro instalar o MacOS para depois instalar o Windows



Para que todo o disco sejá usado devemos usar o "Disk Utility - Utilidade de Disco" para formatar o disco e instalar além de formatar para uso da instalação do MacOS


O disco deve ser formatado em GPT

Formatamos os disco em APFS (pois se trata do novo formato da Apple para seus produtos novos*), particionar o disco em fat16 para depois formatarmos ele para o Windows (O Sistema de arquivos do Windows é NTFS na qual o MacOS não formata).


Devemos agora instalar o MacOS no Disco que foi Formatado para uso do sistema na qual foi Formatado em APFS
